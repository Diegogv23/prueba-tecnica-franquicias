# API de Gestión de Franquicias (Prueba Técnica Juan Diego Gutierrez Venegas)

Esta es una solución robusta basada en **Spring Boot** y **Clean Architecture** para la gestión de franquicias, sucursales y productos.

## Tecnologías Utilizadas
* **Java 17**
* **Spring Boot 3.x** (WebFlux - Programación Reactiva)
* **MongoDB** (Base de datos NoSQL)
* **Docker & Docker Compose** (Contenerización)
* **Lombok** (Reducción de boilerplate)
* **Jakarta Validation** (Validación de datos)
